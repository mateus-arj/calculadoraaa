# Calculadora JavaFX

![Calculadora](imagens/calculadora.png)

Uma calculadora moderna desenvolvida em JavaFX com interface gráfica estilizada, suporte a operações matemáticas ilimitadas e histórico de cálculos.

## 🚀 Funcionalidades

✅ Interface moderna inspirada em calculadoras reais  
✅ Operações ilimitadas  
✅ Soma  
✅ Subtração  
✅ Multiplicação  
✅ Divisão  
✅ Suporte a números decimais  
✅ Histórico de operações  
✅ Botão apagar (⌫)  
✅ Botão limpar (C)  
✅ Visor expansível para contas grandes  
✅ Modo de cálculo da esquerda para a direita  
✅ Modo de cálculo padrão da matemática  
✅ Suporte a parênteses  
✅ Multiplicação implícita (ex: 5(2+3))  
✅ Validação de expressões  
✅ Controle de múltiplos pontos decimais  

## 🖼️ Interface

A aplicação possui:

- Visor escuro moderno
- Botões arredondados
- Operadores destacados em laranja
- Botão de resultado em verde
- Histórico integrado

## 🛠️ Tecnologias Utilizadas

- Java 21
- JavaFX
- Maven
- IntelliJ IDEA

## ▶ Como Executar

1. Clone o repositório:

```bash
git clone URL_DO_SEU_REPOSITORIO
```

2. Abra o projeto no IntelliJ IDEA

3. Aguarde o Maven baixar as dependências automaticamente

4. Execute a classe:

```java
Launcher.java
```
