# Calculadora JavaFX

Uma calculadora moderna desenvolvida em JavaFX com interface gráfica estilizada, suporte a operações matemáticas ilimitadas e histórico de cálculos.

---

## 🚀 Funcionalidades

- ✅ Interface moderna inspirada em calculadoras reais
- ✅ Operações ilimitadas
- ✅ Soma
- ✅ Subtração
- ✅ Multiplicação
- ✅ Divisão
- ✅ Suporte a números decimais
- ✅ Histórico de operações
- ✅ Botão apagar (`⌫`)
- ✅ Botão limpar (`C`)
- ✅ Visor expansível para contas grandes
- ✅ Design moderno em JavaFX

---

## 🖼️ Interface

A aplicação possui:

- visor escuro moderno
- botões arredondados
- operadores destacados em laranja
- botão de resultado em verde
- histórico integrado

---

## 🛠️ Tecnologias utilizadas

- Java
- JavaFX
- Maven
- IntelliJ IDEA
